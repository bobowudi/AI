# Z-Image-Turbo 文生图 AI 工具使用文档

## 1. 工具介绍

### 1.1 概述

Z-Image-Turbo 是一款基于魔塔（ModelScope）平台的 AI 图像生成工具，由通义千问（Tongyi-MAI）团队开发。该工具能够根据文本描述（Prompt）快速生成高质量的图像，支持多种尺寸规格，适用于创意设计、内容创作、概念可视化等多种场景。

### 1.2 核心特性

- **快速生成**：基于 Turbo 架构，生成速度快，响应迅速
- **高质量输出**：支持最高 1024×1024 分辨率的高清图像
- **灵活尺寸**：提供 5 种预设尺寸，满足不同使用需求
- **负面提示词**：支持负面提示词功能，可排除不想要的元素
- **简单易用**：直观的 Web 界面，无需复杂配置即可使用
- **历史记录**：自动保存最近 10 次生成记录，方便回顾和管理

### 1.3 技术架构

- **后端框架**：Node.js + Express
- **前端框架**：Vue.js 3
- **API 服务**：魔塔 ModelScope API
- **模型**：Tongyi-MAI/Z-Image-Turbo

## 2. 工具使用

### 2.1 基本使用流程

#### 步骤一：输入图像描述（Prompt）

在"图像描述"文本框中输入你想要生成的图像内容描述。描述越详细，生成的图像越符合预期。

**示例：**
- `一只可爱的橘猫在阳光下打盹，柔和的光线，高清摄影风格`
- `未来城市的夜景，霓虹灯闪烁，赛博朋克风格，4K 画质`
- `山水画风格的中国传统建筑，水墨画效果，意境深远`

**提示词技巧：**
- 描述主体：明确说明要生成的主要对象
- 添加风格：如"写实风格"、"卡通风格"、"水彩画风格"等
- 描述细节：光线、颜色、构图、氛围等
- 指定质量：如"高清"、"4K"、"专业摄影"等

#### 步骤二：设置负面提示词（可选）

在"负面提示词"文本框中输入你不想在图像中出现的内容，帮助 AI 避免生成不想要的元素。

**示例：**
- `模糊, 低质量, 变形, 扭曲`
- `文字, 水印, 签名`
- `多人, 背景杂乱`

#### 步骤三：选择图片尺寸

从下拉菜单中选择合适的图片尺寸：

- **512 × 512**：小尺寸，适合图标、缩略图
- **768 × 768**：中等尺寸，适合社交媒体
- **1024 × 1024**：大尺寸，适合高质量输出（默认）
- **1024 × 768**：横向，适合横幅、海报
- **768 × 1024**：纵向，适合手机壁纸、竖版海报

#### 步骤四：生成图像

点击"✨ 生成图像"按钮，系统将开始处理你的请求。生成过程通常需要几秒到几十秒不等，请耐心等待。

生成完成后，图像将显示在右侧预览区域，你可以：
- 查看生成的图像
- 查看使用的提示词和尺寸信息
- 点击"⬇️ 下载图片"保存到本地

### 2.2 高级功能

#### 历史记录管理

- 系统自动保存最近 10 次生成记录
- 点击历史记录中的任意图像，可以：
  - 重新查看该图像
  - 自动填充对应的提示词和尺寸设置
  - 基于历史记录进行二次创作

#### 批量生成建议

虽然工具默认每次生成 1 张图像，但你可以：
1. 多次使用相同或相似的提示词，获得不同变体
2. 调整负面提示词，优化生成效果
3. 尝试不同尺寸，找到最适合的规格

### 2.3 使用注意事项

1. **网络连接**：确保网络连接稳定，API 调用需要访问魔塔服务器
2. **提示词长度**：建议提示词长度在 10-200 字之间，过长可能影响效果
3. **生成时间**：复杂场景可能需要更长的生成时间，请耐心等待
4. **API 限制**：请注意 API 调用频率限制，避免频繁请求
5. **版权问题**：生成的图像仅供学习和个人使用，商业用途需注意版权

## 3. 工具优缺点

### 3.1 优点

#### ✅ 生成速度快
- Turbo 架构优化，相比标准模型生成速度更快
- 适合需要快速迭代和尝试的场景

#### ✅ 操作简单
- 直观的 Web 界面，无需安装复杂软件
- 只需输入文字描述即可生成图像
- 支持一键下载，方便保存和使用

#### ✅ 功能完善
- 支持正面和负面提示词，控制更精准
- 多种尺寸选择，适应不同使用场景
- 历史记录功能，方便管理和回顾

#### ✅ 质量稳定
- 基于通义千问团队开发的成熟模型
- 生成的图像质量稳定，细节丰富
- 支持高清输出，满足专业需求

#### ✅ 中文友好
- 对中文提示词支持良好
- 界面完全中文化，使用无障碍
- 适合中文用户的使用习惯

### 3.2 缺点

#### ❌ 依赖网络和 API
- 需要稳定的网络连接
- 依赖第三方 API 服务，可能存在服务中断风险
- API 调用可能有频率限制和费用

#### ❌ 生成结果不可控
- AI 生成具有一定随机性，相同提示词可能产生不同结果
- 复杂场景或特定细节可能无法完全按照预期生成
- 需要多次尝试和调整提示词才能达到理想效果

#### ❌ 功能相对基础
- 不支持图像编辑和后期处理
- 不支持图像到图像的转换（img2img）
- 不支持风格迁移等高级功能

#### ❌ 尺寸限制
- 仅支持预设的 5 种尺寸，不支持自定义尺寸
- 最大分辨率 1024×1024，对于超高分辨率需求可能不足

#### ❌ 无本地部署选项
- 必须通过 API 调用，无法完全离线使用
- 数据隐私完全依赖服务提供商

### 3.3 适用场景

**✅ 推荐使用：**
- 快速概念可视化
- 创意设计和头脑风暴
- 社交媒体内容创作
- 个人项目配图
- 学习和实验 AI 图像生成

**⚠️ 谨慎使用：**
- 需要精确控制细节的商业项目
- 对版权有严格要求的场景
- 需要批量生成大量图像的场景
- 对数据隐私要求极高的场景

## 4. 常见问题（FAQ）

### Q1: 生成失败怎么办？
**A:** 检查网络连接，确认 API 服务正常。如果持续失败，可能是 API 配额用完或服务暂时不可用。

### Q2: 如何提高生成质量？
**A:** 
- 使用更详细和具体的提示词
- 添加风格和质量相关的描述词
- 合理使用负面提示词排除不想要的元素
- 多次尝试，选择最佳结果

### Q3: 生成的图像可以商用吗？
**A:** 需要查看魔塔平台和通义千问的使用条款。建议商业使用前咨询相关法律问题。

### Q4: 支持哪些语言？
**A:** 主要支持中文和英文提示词，其他语言的支持程度可能有限。

### Q5: 可以生成特定人物的图像吗？
**A:** 不建议生成特定真实人物的图像，可能涉及肖像权和隐私问题。
