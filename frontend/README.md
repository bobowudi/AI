# 豆包AI对话前端

这是一个基于 Vue 3 + Vite 的现代化聊天界面，用于与豆包AI模型进行对话。

## 特性

- ✨ 现代化的UI设计
- 💬 实时对话功能
- 📊 Excel文件上传分析
- 🎨 渐变色彩设计
- 📱 响应式布局
- ⚡ 快速的开发体验（Vite）

## 功能说明

### 💬 文本对话
- 在输入框中输入消息
- 按回车键或点击"发送"按钮发送消息
- AI会实时回复

### 📊 Excel文件分析
1. 点击输入框旁边的 📎 按钮
2. 选择 Excel 文件（支持 .xlsx、.xls、.csv 格式）
3. 系统会自动读取并格式化数据
4. 数据会填充到输入框中
5. 点击发送，AI会分析Excel数据并给出见解

**支持的文件格式：**
- .xlsx（Excel 2007+）
- .xls（Excel 97-2003）
- .csv（逗号分隔值）

**分析内容包括：**
- 文件基本信息（工作表名、总行数）
- 列名列表
- 前10行数据预览
- AI智能分析和见解

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vite - 下一代前端构建工具
- Axios - HTTP 客户端
- XLSX - Excel文件解析库

## 开发

安装依赖：

```bash
npm install
# 或
pnpm install
```

启动开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```

## 使用技巧

### Excel数据准备建议
- 确保第一行是列标题
- 数据格式整齐统一
- 避免合并单元格
- 文件大小建议不超过5MB

### 提问技巧
上传Excel后，可以这样提问：
- "帮我总结一下这个数据的主要特点"
- "数据中有什么异常值吗？"
- "请分析数据的趋势"
- "给出数据可视化建议"

## 注意事项

⚠️ 由于前端内存限制，建议上传的Excel文件：
- 行数不超过1000行
- 列数不超过50列
- 文件大小不超过5MB

如果数据量较大，系统只会发送前10行数据供AI分析。
